产品 BOM 列表页设计

"产品BOM管理"的入口页面，展示所有产品 BOM 记录

方案对比

方案一：平铺列表 + 编码链接
BOM编码 产品物料 版本 状态 备注 操作
PBOM-001 台式电脑主机 V1.0 已锁定 首批量产版本 查看 编辑
PBOM-002 笔记本电脑 V1.0 草稿 - 编辑

点击 PBOM-001 编码链接 → 进入 BOM 树编制界面
锁定后编辑按钮禁用，只能查看

✅ 推荐 — 简单直观，符合现有系统风格
方案二：树形列表（内联展开）
▼ 🏭 台式电脑主机 V1.0 已锁定
▼ 🔧 台式电脑半成品 已锁定
📦 主板单元 已锁定
📦 机箱单元 已锁定
▼ 🏭 笔记本电脑 V1.0 草稿
🔧 笔记本半成品 草稿

直接在列表中展开树形结构
适合 BOM 数量少的场景

⚠ 局限性 — 当 BOM 数量多时列表会很冗长，翻页困难

操作入口

新增 BOM：列表页点击"新增"按钮 → 弹出选择框 → 从产品物料中选择（只显示类型为"产品"的物料）→ 系统自动创建 BOM 根节点 → 跳转到编制界面

编辑 BOM：点击编码链接（草稿状态）→ 进入 BOM 编制界面
已锁定的 BOM 只能查看，编码链接仍然可点击

创建新版本：已锁定的 BOM 旁显示"创建新版本"按钮 → 复制整个 BOM 树（version+1）→ 进入编辑

需要确认

列表页使用方案一（平铺列表 + 编码链接进入树编辑），可以吗？
